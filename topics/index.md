# Topics Index

选题总表。所有选题在此统一管理。

## Status Legend

- `captured`：从 inbox 提取，值得展开
- `planned`：确定 brief 和目标日期
- `drafting`：正在写作
- `ready`：完稿待发布
- `published`：已发布
- `dropped`：放弃

## Usage

新增选题时：

1. 在 `topics/` 下创建英文 slug 目录
2. 复制 `_template/topic.md`
3. 在本表中新增一行
4. 填写 brief 和核心观点
