## 风格演化记录

### #1 — 2026-03-31 Agent 数据获取帖

**初稿问题 → 用户修正：**

1. "你最想让 Agent 盯的信息源" → "我日常最多的场景是让 Agent 去盯 X 和 Reddit 上的动态"
   - 信号：不要用"你"假设读者场景，用"我"描述自己的真实需求，更自然

2. "更扎心的发现" → "还有一个发现是"
   - 信号：拒绝 AI 腔的情绪化修饰词，偏好平实的连接词

3. "RSS 桥接能不能救" → "RSS 桥接能不能用"
   - 信号：用词克制，不追求戏剧化表达

4. 列表式分层策略 → 自然叙述串联
   - 信号：社交平台文风，不用 Markdown 列表，用句子自然衔接

5. "发帖购物" → "自动发帖或者回复"
   - 信号：举例要贴近真实使用场景，不要写自己不会做的事

**风格提炼：**
- 语气：口语化但不夸张，陈述事实为主，不煽情
- 用词：平实、克制，拒绝 AI 腔修饰词（扎心、惊喜、一针见血）
- 结构：时间线叙事（我以为→我验证→结论翻转），结尾给判断
- 人称：严格第一人称，从自身经历出发
- 格式：不用列表、不加引号装饰，纯文字流
