# Output Format

每次运行 `daily-discussion-candidates` 时，输出 3-5 个候选方向。

固定格式如下：

```md
## 候选 1
标题：
核心判断：
对应素材：
- [文件名或日期]
- [关键信号]

为什么值得写：

当前实践支撑：

还缺什么验证：

建议下一步：

适合形式：X / 公众号 / 待定
优先级：高 / 中 / 低
```

额外要求：

- `对应素材` 不要罗列过多，控制在 2-4 条
- `核心判断` 必须是一个完整判断句，不是主题词
- `当前实践支撑` 要明确写“已有 / 较弱 / 缺失”
- 如果某个方向不建议继续，也可以输出，但要明确说明原因
